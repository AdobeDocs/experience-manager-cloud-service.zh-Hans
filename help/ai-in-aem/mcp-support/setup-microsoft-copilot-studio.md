# 使用AEM MCP设置Microsoft Copilot Studio {#setup-microsoft-copilot-studio}

按照以下步骤将Microsoft Copilot Studio连接到AEM的MCP服务器。

* 创建新代理。
* 导航到工具部分，然后单击&#x200B;**添加工具**。
* 选择现有工具或创建新工具。
* 配置指向一个或多个AEM MCP服务器URL的新MCP工具。
* 建立可在代理之间共享或专用的连接。
* 重定向后使用您的Adobe ID登录。
* （可选）启用自动确认模式或要求最终用户确认所有工具交互。
* 测试代理时，请先打开连接管理器以向会话分配连接，然后按&#x200B;**重试**。
