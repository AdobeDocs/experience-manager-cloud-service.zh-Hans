# Experience Production Agent概述 {#experience-production-agent}

Experience Production Agent可自动执行高工作量和高容量任务。 增强团队的权能，将耗时数周的手动流程转变为快速的人工智能辅助工作流，让每个体验都保持最新且一致，从而帮助企业实现目标。

## Jobs {#jobs}

代理提供以下作业：

* [内容更新](#content-update)
* [表格创建](#form-creation)
* [通信创建](#communications-creation)
* [站点迁移](#site-migration)

>[!IMPORTANT]
>
>AI生成的响应可能不准确或具有误导性。 请务必仔细检查建议的修复和响应。
>
>另请参阅[Adobe Experience Cloud Generative AI用户指南](https://www.adobe.com/cn/legal/licenses-terms/adobe-dx-gen-ai-user-guidelines.html)。

### 内容更新 {#content-update}

[内容更新](/help/ai-in-aem/agents/production/content-update.md)可以轻松更新整个CMS中的现有内容，包括内容片段、页面、表单和资源。 代理可以执行更新、删除、替换或添加内容元素等操作，以保持体验准确且最新。 输入可以是自然语言描述，在与Jira PDF一起使用时，屏幕截图也可以提供输入。

### 表格创建 {#form-creation}

[表单创建](/help/ai-in-aem/agents/production/form-creation.md)技能使用户能够通过自然语言提示构建自适应表单，而无需依赖开发或IT团队。 此功能可加快表单开发，同时保持品牌一致性，并允许业务用户在不具备深入的技术产品知识的情况下创建表单。

### 通信创建 {#communications-creation}

[通信创建](/help/ai-in-aem/agents/production/communications-creation.md)技能使业务用户能够大规模生成个性化、数据驱动的通信。 从账户报表、政策文件到账单和欢迎套件，Agent将自然语言要求转变为专业通信。

>[!NOTE]
>
> 通信创建技能目前为Alpha版。 如果您想参加，请将您的正式电子邮件地址发送至[aem-forms-ea@adobe.com](mailto:aem-forms-ea@adobe.com)的请求。

### 站点迁移 {#site-migration}

[站点迁移](/help/ai-in-aem/agents/production/site-migration.md)可将非AEM站点无缝迁移到AEM (Experience Delivery Services)环境中，确保它们高性能、符合要求并且可供代理使用。 该代理可简化设置和转换，减少手动工作量和实现价值的时间。
