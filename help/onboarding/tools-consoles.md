# 工具控制台{#tools-consoles}

**工具**&#x200B;控制台让您可以访问一些专用工具，这些工具有助于您管理网站、数字资产及内容存储库的其他方面。

可以从[“全局导航”的“工具”面板](/help/sites-cloud/authoring/getting-started/basic-handling.md#tools-panel)访问这些控制台。
