# Adobe Experience Manager中Cloud Manager作为Cloud Service 2020.9.0 {#release-notes}的发行说明

本页概述了AEM中作为Cloud Service 2020.9.0的Cloud Manager发行说明。

## 发布日期 {#release-date}

AEM中Cloud Manager作为Cloud Service 2020.9.0的发布日期为2020年9月3日。

## 新增功能 {#whats-new-cloud-manager}

* 内容审核已重新标记为Experience Audit。
* 生成过程已分为三个独立的Maven命令。
* 如果无法克隆Git存储库，则最多将重新尝试三次。

### 错误修复 {#bug-fixes-cm}

* “内容审核”选项卡使用作者域而非发布域错误地显示了基本URL。
