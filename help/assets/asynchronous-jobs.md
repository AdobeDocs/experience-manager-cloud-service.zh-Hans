# Asynchronous Jobs {#asynchronous-jobs}

为了减少对性能的不利影响，Adobe Experience Manager(AEM)资产异步处理某些长时间运行且资源密集型资产操作。

有关更多详细信息，请参 [阅“操作](/help/operations/asynchronous-jobs.md) ”用户指南中的文档Asynchronus Jobs。
