# Flash 查看器生命周期终止 {#flash-viewers-end-of-life-notice}

请参阅 [Dynamic Media查看器参考指南](https://experienceleague.adobe.com/docs/dynamic-media-developer-resources.html) 适用于最新支持的查看器。
