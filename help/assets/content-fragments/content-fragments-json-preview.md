# 预览- JSON表示法{#preview-json-representation}

在开发内容片段的模型时，您可能希望视图内容片段的JSON示例输出，如基于模型。 例如，了解最终输出的外观。 在验证模型JSON结构时，这可能会很有帮助，对于每个数据类型的默认示例内容可能会有所帮助。
