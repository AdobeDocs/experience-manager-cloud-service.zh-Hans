# Adobe Experience Manager Assets 云服务中的辅助功能 {#accessibility-in-aem-assets}

Adobe 面向所有用户（包括残障用户）提供产品。Adobe Experience Manager 将不断增强以满足残障用户的需求。Adobe 提供了其产品的合规信息，其中详细说明了适用的标准并概述了辅助功能。有关详细信息，请参阅 AEMaaCS 的 ACR。
