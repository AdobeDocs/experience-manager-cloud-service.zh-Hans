# 获取自定义域名{#getting-cdn}

您可以从域名注册机构、管理和销售域名的公司或组织处购买域名。 域名注册器管理域名系统(DNS)服务器上的域名。

>[!NOTE]
>Cloud Manager不是域名注册机构，不提供DNS服务。

DNS允许我们通过将机器可读的计算机地址（如`184.33.123.64`）转换为易记的地址（如`www.wknd.com`），将自定义的品牌域附加到网站。
