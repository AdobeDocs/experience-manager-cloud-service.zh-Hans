# 查看和更新并替换自定义域名{#view-update-replace-cdn}

使用“查看”和“更新”菜单，只需查看一个或多个自定义域名的详细信息即可。
或者，您也可以更改用于自定义域名的SSL证书。

>[!NOTE]
>SSL证书必须有效，并包含自定义域名。
