# 查看和更新并替换自定义域名{#view-update-replace-cdn}

使用“视图和更新”菜单，只需视图一个或多个自定义域名的详细信息。
或者，也可以更改要用于自定义域名的SSL证书。

>[!NOTE]
>SSL证书必须有效并包含自定义域名。
