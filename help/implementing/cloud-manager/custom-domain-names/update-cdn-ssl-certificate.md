# 更新自定义域名的SSL证书 {#update-ssl-certificate-cdn}

可以从UI中的两个不同位置执行更新操作：

1. 导航到 **域设置** 页面。
1. 或者，从导航到特定的环境详细信息页面 **环境** ，然后导航到“域名”表。
   >[!NOTE]
   >确定列出您要查看或更新的自定义域名的行。
1. 选择 **...** 菜单
1. 选择 **查看和更新** 选项。
1. 根据需要更改SSL证书并确认您的提交。
