# 更新自定义域名的SSL证书{#update-ssl-certificate-cdn}

更新操作可以从UI的两个不同位置执行：

1. 从“环境”屏幕导航到&#x200B;**域设置**&#x200B;页。
1. 或者，也可以从&#x200B;**环境**&#x200B;屏幕导航到特定的环境详细信息页面，然后导航到域名表。
   >[!NOTE]
   >确定要视图或更新的自定义域名列在的行。
1. 选择&#x200B;**...**&#x200B;菜单
1. 选择&#x200B;**视图和更新**&#x200B;选项。
1. 根据需要更改SSL证书并确认您的提交。
