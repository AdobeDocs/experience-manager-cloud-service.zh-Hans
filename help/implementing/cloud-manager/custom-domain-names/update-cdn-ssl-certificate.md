# 更新自定义域名的SSL证书{#update-ssl-certificate-cdn}

可以从UI中的两个不同位置执行更新操作：

1. 从“环境”屏幕中，导航到&#x200B;**域设置**&#x200B;页面。
1. 或者，从&#x200B;**Environments**&#x200B;屏幕中导航到特定的“环境详细信息”页面，然后导航到“域名”表。
   >[!NOTE]
   >确定列出您要查看或更新的自定义域名的行。
1. 选择&#x200B;**...**&#x200B;菜单
1. 选择&#x200B;**查看并更新**&#x200B;选项。
1. 根据需要更改SSL证书并确认您的提交。
