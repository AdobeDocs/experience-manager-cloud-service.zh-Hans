# 删除 IP 允许列表 {#delete-allow-list}

按照以下步骤删除IP允许列表:

1. 导航到 **域设置** 页面 **环境** 屏幕。
1. 确定要删除的自定义域名在其中列出的行。
1. 选择 **...** 菜单。
1. 选择 **删除** 选项。
1. 确认您的提交。

>[!NOTE]
>删除IP允许列表将自动取消应用该IP中的所有服务，并将其从表中删除。
