# 删除IP允许列表{#delete-allow-list}

请按照以下步骤删除IP允许列表:

1. 从“环境”屏幕导航到“域设置”页。
1. 确定要删除的自定义域名所在的行。
1. 选择&#x200B;**...**&#x200B;菜单。
1. 选择删除选项。
1. 确认提交。

>[!NOTE]
>删除IP允许列表将自动取消应用该IP的所有服务，并将其从表中删除。
