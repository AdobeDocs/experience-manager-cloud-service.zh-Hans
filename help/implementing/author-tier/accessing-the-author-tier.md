# 访问创作层 {#accessing-the-author-tier}

通过标准且不可自定义的URL，访问AEMas a Cloud Service环境的创作层。

URL格式如下：

* `https://author-p[Program ID]-e[EnvironmentID].adobeaemcloud.com/`

不支持AEM创作层的自定义域名。
