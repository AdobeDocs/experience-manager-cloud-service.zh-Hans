# 在Cloud Acceleration Manager中进入实时阶段 {#golive-phase-cam}

“上线”卡提供了所有相关内容，其中重点介绍了准备工作、要采取的具体操作，并进行了充分考虑，以确保AEMas a Cloud Service顺利成功上线。

单击您的项目卡片以打开项目登录页面，然后导航到 **上线** 部分，如下图所示。
