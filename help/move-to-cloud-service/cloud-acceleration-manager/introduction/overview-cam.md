# Cloud Acceleration Manager {#overview-cam}概述

Cloud Acceleration Manager为客户提供了一站式迁移体验，通过提供初始评估，指导客户使用正确的工具、文档和最佳实践，确保在AEM as a Cloud Service中成功上线，从而指导客户完成整个迁移过程。
