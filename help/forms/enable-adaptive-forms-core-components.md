# 检查和启用自适应Forms核心组件 {#enable-headless-adaptive-forms-on-aem-forms-cloud-service}

| -------- | ---------------------------- |

## 自适应Forms核心组件可用性

所有&#x200B;**AEM Forms as a Cloud Service**&#x200B;环境都支持使用核心组件创建自适应Forms。 有关表单创建的分步说明，请参阅[创建自适应表单](/help/forms/create-an-adaptive-form.md)。

### 为Forms 6.5启用自适应AEM核心组件

对于&#x200B;**AEM 6.5 Forms**&#x200B;和&#x200B;**AEM 6.5 LTS Forms**&#x200B;环境，默认情况下不启用自适应Forms核心组件。 您必须使用AEM原型将自适应Forms功能添加到您的环境中。

请遵循相关指南：
- 对于AEM 6.5 Forms：[在AEM 6.5 Forms上启用自适应Forms核心组件](https://experienceleague.adobe.com/docs/experience-manager-65/forms/adaptive-forms-core-components/enable-adaptive-forms-core-components.html)。
- 对于AEM 6.5 LTS Forms：[在AEM 6.5 LTS Forms上启用自适应Forms核心组件](https://experienceleague.adobe.com/en/docs/experience-manager-65-lts/content/forms/adaptive-forms-core-components/enable-adaptive-forms-core-components)。

**注意：**&#x200B;如果在旧版AEM 6.5环境中启用自适应Forms核心组件的任何步骤不明确或存在特定于环境的先决条件，请在尝试启用之前澄清这些要求。
