### 已知问题 {#known-issues-16357}

* 某些用户在上创建表单时可能会遇到问题 `Cloud Service version 2024.5.16461`，问题是，当用户创建表单时，创建对话框中会弹出以下错误消息：

  `A server error occurred. Try again after sometime.`

  出现此问题的原因是，作者发布表单时没有先发布表单中使用的模板，这会导致 `jcr:uuid` 和其他受保护及系统生成的资产转入 `<template-path>/initial/jcr:content` 节点。 这会导致后续表单创建失败。

  要解决此问题，请按照 [解决方法步骤](/help/forms/form-creation-failing.md#workaround-resolution-form-creation-fails).

### 已弃用的功能和 API {#deprecated-16145}

* [在 Adobe Developer Console 中弃用 JWT 凭据](/help/security/jwt-credentials-deprecation-in-adobe-developer-console.md)

* 自 2024 年 5 月 1 日起，Adobe Dynamic Media 将终止对以下内容的支持：

   * SSL（安全套接字层）2.0
   * SSL 3.0
   * TLS（传输层安全性）1.0 和 1.1
   * TLS 1.2 中的以下弱密码：
